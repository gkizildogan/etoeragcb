# P10 golden set v1

This synthetic corpus and its labels are dedicated to the public domain under
CC0-1.0. It contains no production documents, user prompts, feedback, or
personal data.

The set is intentionally small and diagnostic. It covers English and Turkish
queries, exact identifiers, headings, collection/document scopes, semantic
paraphrases, repeated text on distinct pages, ambiguous document hints,
combined web/document evidence, and related as well as unrelated
unanswerable questions.

Relevance grades are human-authored:

- `3`: directly and completely supports the requested fact;
- `2`: supports the same fact in another language or older comparison source;
- `1`: partially useful context.

This is a project regression/calibration set, not a claim of general-domain
retrieval quality. P11 may add larger external benchmarks, but they must not
replace these metadata and safety cases.
